# Learner Experience Analysis Dashboard

## Overview
This project explores learner experience data to uncover patterns behind registration, engagement, and completion.  
The goal is to answer key business questions and provide insights that support better decision-making.

## Objectives
- Understand why learners don’t start or complete courses  
- Analyze demand across opportunities, categories, and countries  
- Identify drop-off points in the learner journey  
- Build clear KPIs for performance tracking  

## Process
- Data Cleaning and Preparation (team collaboration)  
- Data Modeling using Star Schema  
- Defining Business Questions and KPIs  
- Data Analysis and Insight Extraction  
- Dashboard Design (AI-assisted for layout and visualization)  

## Key Insights
- 8,558 total registrations across 23 opportunities  
- 71 countries reached, with high concentration in the US and India  
- Internships represent the highest demand (approximately 63%)  
- Only 9% of users start after registration  
- 41.7% rejection rate, indicating a mismatch between applicants and course requirements  

## Tools Used
- Excel / Power BI  
- Power Query  
- Data Modeling (Star Schema)  
- AI (for dashboard design support)
